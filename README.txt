# ОППО АГУ — мини‑сайт
Готовая тёмная тема с фирменными цветами (#0039a6, #d51313, #5271ff, #ffffff) и шрифтом Vela Sans (кириллица).

## Файлы
- `index.html` — главная: расписание + 3 карточки с модальными окнами.
- `roulette.html` — страница с рулеткой. Вероятность выигрыша — 10%.
- `styles.css` — стили.
- `script.js` — логика модальных окон.
- `assets/fonts/` — положите сюда файлы шрифта `VelaSans.woff2` / `VelaSans.woff`.

## Запуск
Откройте `index.html` в браузере. Для рулетки: `roulette.html`.
